# Prompt Edit ‚Äî Landing Page

A modern, single-page marketing site for an AI-tools marketplace, built with **Vue 3**, **Vite**, and **Tailwind CSS**. The page is fully responsive, animation-rich, and built from small, swappable section components so the copy and visuals can be updated quickly.
---

## Tech stack

| Layer | Choice | Why |
| --- | --- | --- |
| Framework | **Vue 3** (`<script setup>`, Composition API) | Lightweight, ergonomic SFCs |
| Build tool | **Vite 5** | Instant HMR, tiny prod bundles |
| Styling | **Tailwind CSS 3** + `@layer` components | Utility-first, brand-token driven |
| Animations | Custom CSS + `IntersectionObserver` directives | Zero runtime animation library |
| Tooling | PostCSS, Autoprefixer | Standard PostCSS pipeline |

No runtime UI library, no state-management dependency, no router ‚Äî it's a single landing page kept intentionally small (~108 npm packages installed).

## Quick start

### Prerequisites

- **Node.js 18+** (LTS recommended) ‚Äî verify with `node --version`
- **npm 9+** ships with Node.js

### Install & run

```powershell
git clone <your-repo-url>
cd prompt-edit-clone
npm install
npm run dev
```

Open the URL Vite prints (typically `http://localhost:5173/`).

### Available scripts

| Script | What it does | Output |
| --- | --- | --- |
| `npm run dev` | Dev server with hot module replacement | in-memory, `localhost:5173` |
| `npm run build` | Optimized production build (minify, tree-shake, hash) | `dist/` |
| `npm run preview` | Serves the production build for verification | `localhost:4173` |

## Section components

Each `.vue` file under `src/components/` is fully self-contained ‚Äî its content is a plain JS array at the top of `<script setup>`, so non-developers can edit copy without touching template logic.

| Component | Role |
| --- | --- |
| `NavBar.vue` | Sticky top nav with scroll-state styling |
| `HeroSection.vue` | Headline, CTAs, floating preview cards, logo marquee |
| `IntroSection.vue` | Old way vs. new way comparison + benefit cards |
| `ToolsSection.vue` | Tabbed bento for Image / Video / Audio tools |
| `EditingPlugins.vue` | Plugin pitch + mock editor panel |
| `PricingSection.vue` | Two-plan pricing with featured plan highlight |
| `TemplatesLibrary.vue` | Stats showcase with count-up animation |
| `AssetCategories.vue` | Six asset-category cards + interactive SFX demo |
| `Testimonials.vue` | Quote cards + trust-stat strip |
| `WhoIsItFor.vue` | Pain-point checklist |
| `Guarantee.vue` | Animated 30-day guarantee badge |
| `FaqSection.vue` | Accordion FAQ |
| `FooterSection.vue` | CTA banner, nav, legal disclaimers |

To re-order, hide, or duplicate sections, edit `src/App.vue`:

```vue
<main>
  <HeroSection />
  <IntroSection />
  <!-- drop or rearrange any section here -->
</main>
```
